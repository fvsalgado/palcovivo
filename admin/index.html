Templates de Scraper por Tipo

Após adicionar o venue, cria o ficheiro do scraper em:

pipeline/scrapers/{venue-id}/scraper.py

Segue o padrão do scraper CCB. A função run() deve retornar uma lista de eventos raw.

Os scrapers são descobertos automaticamente — basta criar a pasta com scraper.py.
